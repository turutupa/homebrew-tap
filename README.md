# Homebrew Tap for Yames

This is the official [Homebrew](https://brew.sh) tap for [Yames](https://turutupa.github.io/yames/) — Yet Another Metronome Everyone Skips.

## Install

```bash
brew install --cask turutupa/tap/yames
```

## Update

```bash
brew upgrade --cask yames
```

## What is Yames?

A free, open-source desktop metronome with an always-on-top floating widget, sub-millisecond audio precision, zen mode, drill practice, and 10 beautiful themes. Built with Rust + Tauri v2.

- Homepage: https://turutupa.github.io/yames/
- Source: https://github.com/turutupa/yames
